Keyloggers are used by malicios threat actors to collect the key stroke input from a victim's computer

This is a simple of a keyboard keylogger

-> Run the main.py file and keep it runnning , now write anything anywhere in your computer 

-> The log.txt file will store all the keystrokes including ctrl,shift,backspace

-> The log 2.txt file will store keystrokes excluding ctrl,shift,backspace

-> Running the clear_history.py will delete all previous key strokes

-> NOTE - listenMouse.py and control.py is just for learning the pynput library purpose and isn't related to this project functionality.
